# PHP SDK Release Notes

- CloudControl adds `CreateResource`, `DeleteResource`, `DescribeResourceType`, `GetResource`, `GetTask`, `ListResources`, `ListResourceTypes`, `ListTasks`, and `UpdateResource`.
- TIS adds `MigrateDeviceWithoutApproval`.
- CloudMonitor updates `GetMetricData` and `ListAlertGroup`.
- I18N OpenAPI adds `VideoEditorDeleteSubtitle`, `VideoProjectSubtitleDelete`, and `VideoProjectVideoDelete`, and updates `VideoEditorDeleteSubtitle`, `VideoEditorGetEmotionTags`, `VideoProjectSerialDubTaskCreate`, and `VideoProjectSerialTaskCreate`.
